# Site do Núcleo dos Estudantes da UCM

Site oficial do Núcleo dos Estudantes da Universidade Católica de Moçambique.

## Estrutura do Projeto

## Funcionalidades

1. **Design Responsivo**
   - Adapta-se a todos os dispositivos (mobile, tablet, desktop)
   - Menu mobile com toggle

2. **Seções do Site**
   - Página inicial com hero section
   - Sobre o núcleo
   - Atividades realizadas
   - Direção atual
   - Últimas notícias
   - Formulário de contacto

3. **Funcionalidades JavaScript**
   - Formulário de contacto com validação
   - Carregamento dinâmico de notícias
   - Scroll suave
   - Sistema de notificações
   - Efeitos de scroll

4. **Recursos Visuais**
   - Paleta de cores da UCM (azul e amarelo)
   - Animações suaves
   - Ícones FontAwesome
   - Imagens otimizadas

## Personalização

### 1. Alterar Cores
No arquivo `css/style.css`, modifique as variáveis CSS na raiz:
```css
:root {
    --primary-color: #0056a9;    /* Azul principal */
    --secondary-color: #003366;  /* Azul escuro */
    --accent-color: #ffcc00;     /* Amarelo */
    /* ... */
}
